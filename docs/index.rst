heatrapy documentation
======================

Heatrapy stands for HEAt TRAnsfer in PYthon. It simulates dynamic 1D
and 2D heat transfer processes in solids using the finite difference
method. Heatrapy includes both the modeling of caloric effects and the
incorporation of phase transitions.
